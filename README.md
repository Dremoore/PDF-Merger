# Free PDF Merger

This is a fully free web app to:

- Upload multiple PDF files
- Drag and drop to reorder
- Merge PDFs in chosen order
- Download merged PDF
- Print merged PDF directly from browser

**Features:**

- Works offline in the browser
- No server required
- Fully free and open-source
- Hostable on GitHub Pages

**How to use:**

1. Open `index.html` in your browser (or your GitHub Pages link)
2. Upload PDF files
3. Reorder files if needed
4. Click **Merge & Download** to get a single PDF
5. Click **Print Merged PDF** to print directly
